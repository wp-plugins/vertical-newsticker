== Description ==

Creates a vertical newsticker from any feed.

Options include: News source and scroll speed

Style fully editable in stylesheet.

== Installation ==

Download Vertical Newsticker and install.

Display options are found at Settings -> Vertical Newsticker.

* Choose your news source (installation default http://clearcrest.net/wp/vt-news-en)
* Scroll speed
